# homebrew-drifty

Homebrew tap for [Drifty](https://drifty.so), a Mac-focused AI time tracker.

## Install

```sh
brew install jeongjin0/drifty/drifty
```

This taps the formula repo and installs the `drifty` cask, which downloads the official
Drifty.app release for Apple Silicon Macs from the official releases repo
(edgethink00/drifty_releases).

## Uninstall

```sh
brew uninstall --cask drifty
```

## Notes

This is an unofficial personal tap maintained alongside the Drifty project to make
`brew install` work today. It is not yet submitted to homebrew-core/homebrew-cask.
